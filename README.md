# FairMarket AI: A Privacy-Preserving Federated Framework for Detecting Digital Redlining 

## Problem Statement
E-commerce platforms may show different prices based on user location, login status, device, or browsing history. This creates unfair and non-transparent pricing.

## Solution
This Chrome extension detects pricing bias by comparing product prices across:
- Logged-in vs logged-out users
- Different delivery locations
- Different devices (mobile vs desktop)

## Tech Stack
- JavaScript
- Chrome Extension APIs
- HTML / CSS
- GitHub

## How It Works
1. User visits a product page
2. Extension extracts product price
3. Compares prices across sessions or conditions
4. Flags potential pricing bias

## Installation
1. Clone this repo
2. Open Chrome → Extensions → Developer Mode
3. Load unpacked → select the project folder
